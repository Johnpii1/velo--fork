# Support

If you need help with Velo, there are several ways to get support.

## Documentation

Start with the repository documentation in the docs/ directory and the project README.

## Issues

Use GitHub Issues for bug reports, feature requests, and general questions that are appropriate for public discussion.

## Security

For vulnerability reports, follow the process in [SECURITY.md](SECURITY.md).

## Community

If you are looking for broader discussion or design feedback, use GitHub Discussions once enabled for the repository.
